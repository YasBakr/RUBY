Módulo 1: Introdução ao Ruby

Bem-vindo ao Módulo 1! Aqui você aprenderá os conceitos básicos da linguagem Ruby, incluindo a sintaxe, variáveis e estruturas de controle. Vamos começar com uma breve introdução ao que é Ruby.

O que é Ruby?

Ruby é uma linguagem de programação interpretada, dinâmica e orientada a objetos. Foi criada no Japão por Yukihiro Matsumoto, também conhecido como "Matz", em meados dos anos 1990. Ruby é conhecida por sua simplicidade e produtividade, sendo amplamente utilizada para o desenvolvimento web com o framework Ruby on Rails.

Sintaxe Básica

A sintaxe de Ruby é bastante intuitiva e fácil de entender. Aqui estão alguns exemplos básicos para você se familiarizar: 

# Exemplo de um "Olá, Mundo!" em Ruby
puts 'Olá, Mundo!'

# Operações matemáticas básicas
soma = 2 + 2
puts soma  # Saída: 4

Variáveis e Tipos

Ruby tem um sistema de tipos dinâmico, o que significa que você não precisa especificar o tipo da variável ao declará-la. Abaixo estão exemplos de diferentes tipos de variáveis: 

# Inteiros
idade = 25

# Strings
nome = "Maria"

# Float (número decimal)
altura = 1.75

# Booleano
estudante = true

Estruturas de Controle

Ruby oferece várias estruturas de controle, como condicionais e laços de repetição. Aqui estão alguns exemplos de como utilizá-los: 

# Condicional if-else
idade = 18
if idade >= 18
  puts 'Você é maior de idade.'
else
  puts 'Você é menor de idade.'
end

# Laço de repetição while
i = 0
while i < 5
  puts "i vale #{i}"
  i += 1
end

Conclusão

Você aprendeu o básico da linguagem Ruby, incluindo sua sintaxe, variáveis e estruturas de controle. No próximo módulo, você aprenderá mais sobre métodos, classes e outros conceitos importantes da programação em Ruby.
