Módulo 2: Instalação e Primeiro Programa

No Módulo 2, vamos aprender como instalar o Ruby no seu sistema operacional e criar seu primeiro programa em Ruby. Este é o primeiro passo prático para começar a programar!

Instalando o Ruby

A instalação do Ruby é simples e pode ser feita em diferentes sistemas operacionais. Abaixo estão as instruções para Windows, macOS e Linux:

Windows

Para instalar o Ruby no Windows, siga estas etapas: 

1. Acesse o site oficial: rubyinstaller.org.
2. Baixe o instalador recomendado.
3. Siga as instruções do instalador para concluir a instalação.

macOS

No macOS, Ruby já vem pré-instalado. No entanto, recomenda-se usar um gerenciador de versões como o rbenv: 

# Instale o rbenv via Homebrew:
brew install rbenv

# Configure o rbenv:
rbenv init

# Instale a versão mais recente do Ruby:
rbenv install 3.1.0

Linux (Ubuntu)

No Ubuntu, você pode instalar o Ruby usando o gerenciador de pacotes apt: 

# Atualize seus pacotes:
sudo apt update

# Instale o Ruby:
sudo apt install ruby-full

Seu Primeiro Programa em Ruby

Agora que o Ruby está instalado, vamos criar nosso primeiro programa! Vamos escrever um simples programa "Olá, Mundo!" em Ruby. 

# Abra seu editor de texto e digite o seguinte código:
puts 'Olá, Mundo!'

# Salve o arquivo como 'ola_mundo.rb' e execute no terminal:
ruby ola_mundo.rb

Parabéns! Você acabou de criar e executar seu primeiro programa em Ruby.

Conclusão
