Módulo 4: Projeto Final em Ruby

Bem-vindo ao módulo final! Agora vamos aplicar o que aprendemos até aqui para criar um projeto mais complexo que usa vetores, matrizes e manipulação de arquivos. Este projeto será uma simulação de um sistema de cadastro e consulta de notas de alunos.

Projeto: Sistema de Notas

Neste projeto, vamos criar um programa que permite o cadastro de notas de alunos em várias disciplinas, armazenar esses dados em uma matriz e salvar os resultados em um arquivo. O programa também permitirá consultar as notas de um aluno específico.

Etapas do Projeto

1. O programa solicitará o nome dos alunos e as notas de três disciplinas. 2. Armazenaremos esses dados em uma matriz, onde cada linha representa um aluno e as colunas representam suas notas. 3. Após o cadastro, os dados serão salvos em um arquivo chamado notas_alunos.txt. 4. O usuário poderá consultar as notas de um aluno específico digitando seu nome.

Código do Sistema de Notas

# Definindo um array para armazenar os dados dos alunos
alunos = []

# Número de alunos e disciplinas
num_alunos = 3
disciplinas = ["Matemática", "Física", "Programação"]

# Coletando as notas dos alunos
num_alunos.times do |i|
  puts "Digite o nome do aluno #{i+1}:"
  nome = gets.chomp

  # Inicializando o array de notas do aluno
  notas = []

  # Coletando as notas para cada disciplina
  disciplinas.each do |disciplina|
    puts "Digite a nota de #{nome} em #{disciplina}:"
    nota = gets.chomp.to_f
    notas << nota
  end

  # Armazenando o nome e as notas na matriz
  alunos << { nome: nome, notas: notas }
end

# Salvando os dados em um arquivo
File.open("notas_alunos.txt", "w") do |file|
  alunos.each do |aluno|
    file.puts "Aluno: #{aluno[:nome]}"
    disciplinas.each_with_index do |disciplina, index|
      file.puts "#{disciplina}: #{aluno[:notas][index]}"
    end
    file.puts "----------------------"
  end
end

# Função para consultar notas de um aluno específico
def consultar_notas(alunos, disciplinas)
  puts "Digite o nome do aluno para consultar as notas:"
  nome_busca = gets.chomp

  aluno_encontrado = alunos.find { |aluno| aluno[:nome].casecmp?(nome_busca) }

  if aluno_encontrado
    puts "Notas de #{aluno_encontrado[:nome]}:"
    disciplinas.each_with_index do |disciplina, index|
      puts "#{disciplina}: #{aluno_encontrado[:notas][index]}"
    end
  else
    puts "Aluno não encontrado."
  end
end

# Executando a consulta
consultar_notas(alunos, disciplinas)
