Módulo 3: Vetores, Matrizes e Arquivos em Ruby

No Módulo 3, vamos mergulhar em conceitos importantes de Ruby, como a criação e manipulação de vetores, matrizes e arquivos. Esses são elementos cruciais para trabalhar com dados de forma eficiente.

Vetores em Ruby

Vetores, ou arrays, são uma estrutura de dados essencial para armazenar coleções de itens. Abaixo está um exemplo simples de como criar e manipular um vetor em Ruby. 

# Criando um vetor
meu_array = [1, 2, 3, 4, 5]

# Acessando elementos
puts meu_array[0]  # Exibe o primeiro elemento: 1

# Adicionando um novo elemento
meu_array << 6

# Exibindo o vetor atualizado
puts meu_array.inspect

Matrizes em Ruby

Matrizes são arrays multidimensionais. Elas são úteis quando você precisa representar dados em várias dimensões, como em tabelas. Veja um exemplo de como criar e acessar uma matriz: 

# Criando uma matriz 2x2
matriz = [[1, 2], [3, 4]]

# Acessando um elemento específico
puts matriz[1][0]  # Exibe o valor 3

# Exibindo a matriz completa
matriz.each { |linha| puts linha.inspect }

Manipulação de Arquivos em Ruby

A manipulação de arquivos em Ruby é simples e direta. Você pode abrir, ler, escrever e fechar arquivos facilmente. Abaixo, um exemplo de como criar e escrever em um arquivo: 

# Criando e escrevendo em um arquivo
File.open('exemplo.txt', 'w') do |arquivo|
  arquivo.puts 'Este é um exemplo de escrita em arquivo em Ruby.'
end

# Lendo o conteúdo do arquivo
conteudo = File.read('exemplo.txt')
puts conteudo
